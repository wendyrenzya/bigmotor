Tutorial Print Struk Bengkel
1. Install aplikasi BigMotor Printer.
Jika belum punya, download di sini.
2. Setelah aplikasi terinstall.
Aplikasi tidak muncul di home karena berfungsi sebagai background service.
3. Hubungkan HP dengan bluetooth printer lalu tekan tombol print pada halaman struk.
4. Pilih aplikasi printer pada Android Share.
Jika tidak muncul, geser ke kanan lalu tekan menu lainnya.
5. Pilih ikon robot hijau bertuliskan BigMotorPrinter.
6. Akan tampil preview printer.
Tekan tombol biru Print.
7. Pastikan printer terhubung Bluetooth.
Jika terhubung akan muncul nama printer berwarna hijau.
8. Tekan tombol print dan tunggu struk keluar.
